# 🚨 Node.js O'RNATISH KERAK!

## ⚠️ Muammo
Sizning kompyuteringizda **Node.js** o'rnatilmagan. 

---

## ✅ Yechim: Node.js o'rnating

### Step 1: Node.js vebsaytiga boring
👉 **https://nodejs.org/** saytiga oching

### Step 2: LTS versiyasini yuklab oling
- "LTS" yozuvini tanlang (Masalan: v18.x yoki v20.x)
- **Windows Installer (.msi)** ni tanlang
- "Download" bosing

### Step 3: Installer'ni ishga tushiring
1. Yuklab olingan `.msi` faylni ochang
2. Wizard'da "NEXT" bosing va "Install" bosing
3. Korona bersovni kiritishingiz mumkin bo'ladi
4. Xalos bo'lgach, kompyuterni RESTART qiling

### Step 4: Tekshiring Node.js o'rnatilganligini
Windows Start menusida **PowerShell** yoki **Command Prompt (cmd)** oching va yozing:

```
node --version
npm --version
```

Agar raqamlar chiqsa (masalan `v20.10.0`), demak muvaffaqiyat! ✅

---

## 🚀 Keyin qiladigan ishlar

Endi qayturib qo'l qo'ymang:

1. Papkaga boring:
```
cd Desktop\oybek\MANSUR SHASHLIK
```

2. Paketlarni o'rnating:
```
npm install
```

3. Serverni boshlang:
```
npm start
```

4. Brauzer'da oching:
```
http://localhost:3000
```
